# Capability Library Frontend Demo

该独立 Vite 工程只包含后端能力组件库对应的两个审核界面。

## 启动

```bash
npm ci
npm run dev
```

打开：

- `http://127.0.0.1:5173/music-education-review.html`
- `http://127.0.0.1:5173/virtual-instrument-review.html`

## 验证

```bash
npm test
npm run build
npm run preview
```

页面状态仅保存在浏览器内存中。工程不包含环境文件、服务凭据、用户上传、数据库或个人数据。

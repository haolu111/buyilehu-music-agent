# 互动包全覆盖测试教案

本目录用于测试“教案解析 → Agent 选择 → 互动包生成 → 教师预览 → 发布课堂”的完整流程。

## 使用方法

1. 在教师端进入“上传教案”。
2. 每次上传一份 `.txt` 教案并确认解析结果。
3. 生成互动包时，建议选择 35–45 分钟、标准或丰富密度。
4. 对照 `互动覆盖矩阵.md` 检查 Agent 的选择及推荐理由。
5. Agent 会依据目标自主取舍，因此一份教案不保证一次命中全部候选；未命中的活动可再次生成，或使用对应的专项教案。

## 文件说明

- 01：节拍、律动与节奏问答
- 02：歌词节奏、分句学唱与合唱发声
- 03：身体打击、图形谱、曲式和音条琴创编
- 04：听赏导入、主题再现、音色与乐器分类
- 05：唱名、旋律线、简谱与视唱练耳
- 06：奥尔夫合奏、分声部、指挥、展示评价与出口票
- 07：音乐游戏专项
- 08：虚拟乐器专项

前六份覆盖正式注册表中的 33 个活动；后两份用于验证游戏和虚拟乐器节点。
